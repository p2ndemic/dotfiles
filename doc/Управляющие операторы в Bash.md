### **`&` vs `&&`** (—É–ø—Ä–∞–≤–ª—è—é—â–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤ Bash)

1. **`&`** (–∞–º–ø–µ—Ä—Å–∞–Ω–¥ –≤ –∫–æ–Ω—Ü–µ –∫–æ–º–∞–Ω–¥—ã)  
   - **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç**: –ó–∞–ø—É—Å–∫–∞–µ—Ç –∫–æ–º–∞–Ω–¥—É –≤ **—Ñ–æ–Ω–æ–≤–æ–º —Ä–µ–∂–∏–º–µ**  
   - **–ü—Ä–∏–º–µ—Ä**:  
     ```bash
     xdg-open file.txt &  # –§–∞–π–ª –æ—Ç–∫—Ä–æ–µ—Ç—Å—è, –∞ —Å–∫—Ä–∏–ø—Ç –ø—Ä–æ–¥–æ–ª–∂–∏—Ç —Ä–∞–±–æ—Ç—É
     ```
   - **–ó–∞—á–µ–º**: –ß—Ç–æ–±—ã —Å–∫—Ä–∏–ø—Ç –Ω–µ –∂–¥–∞–ª –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è –∫–æ–º–∞–Ω–¥—ã  
   - **–ê–Ω–∞–ª–æ–≥**: –ö–∞–∫ –µ—Å–ª–∏ –±—ã –≤—ã –∑–∞–ø—É—Å—Ç–∏–ª–∏ –ø—Ä–æ–≥—Ä–∞–º–º—É –∏ —Å—Ä–∞–∑—É —Å–º–æ–≥–ª–∏ —Ä–∞–±–æ—Ç–∞—Ç—å –≤ —Ç–µ—Ä–º–∏–Ω–∞–ª–µ –¥–∞–ª—å—à–µ

2. **`&&`** (–¥–≤–æ–π–Ω–æ–π –∞–º–ø–µ—Ä—Å–∞–Ω–¥ –º–µ–∂–¥—É –∫–æ–º–∞–Ω–¥–∞–º–∏)  
   - **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç**: –í—ã–ø–æ–ª–Ω—è–µ—Ç –≤—Ç–æ—Ä—É—é –∫–æ–º–∞–Ω–¥—É **—Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –ø–µ—Ä–≤–∞—è —É—Å–ø–µ—à–Ω–æ –∑–∞–≤–µ—Ä—à–∏–ª–∞—Å—å** (–∫–æ–¥ –≤–æ–∑–≤—Ä–∞—Ç–∞ 0)  
   - **–ü—Ä–∏–º–µ—Ä**:  
     ```bash
     mkdir my_dir && cd my_dir  # –ü–µ—Ä–µ–π–¥–µ—Ç –≤ –ø–∞–ø–∫—É —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –æ–Ω–∞ —Å–æ–∑–¥–∞–ª–∞—Å—å
     ```
   - **–õ–æ–≥–∏–∫–∞**: "–ò –µ—Å–ª–∏ –ø—Ä–µ–¥—ã–¥—É—â–∞—è –∫–æ–º–∞–Ω–¥–∞ —É–¥–∞–ª–∞—Å—å"

---

### –î—Ä—É–≥–∏–µ –≤–∞–∂–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

3. **`||`** (–¥–≤–æ–π–Ω–∞—è –≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–∞—è —á–µ—Ä—Ç–∞)  
   - **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç**: –í—ã–ø–æ–ª–Ω—è–µ—Ç –≤—Ç–æ—Ä—É—é –∫–æ–º–∞–Ω–¥—É **—Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –ø–µ—Ä–≤–∞—è –∑–∞–≤–µ—Ä—à–∏–ª–∞—Å—å —Å –æ—à–∏–±–∫–æ–π**  
   - **–ü—Ä–∏–º–µ—Ä**:  
     ```bash
     ping google.com || echo "–ò–Ω—Ç–µ—Ä–Ω–µ—Ç –æ—Ç–∫–ª—é—á–µ–Ω!"
     ```

4. **`;`** (—Ç–æ—á–∫–∞ —Å –∑–∞–ø—è—Ç–æ–π)  
   - **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç**: –í—ã–ø–æ–ª–Ω—è–µ—Ç –∫–æ–º–∞–Ω–¥—ã **–ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ**, –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç —É—Å–ø–µ—Ö–∞ –ø—Ä–µ–¥—ã–¥—É—â–µ–π  
   - **–ü—Ä–∏–º–µ—Ä**:  
     ```bash
     echo "Start"; ls; echo "End"  # –í—ã–≤–µ–¥–µ—Ç –≤—Å–µ —Ç—Ä–∏ —Å—Ç—Ä–æ–∫–∏ –ø–æ–¥—Ä—è–¥
     ```

5. **`|`** (–≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–∞—è —á–µ—Ä—Ç–∞)  
   - **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç**: –ü–µ—Ä–µ–Ω–∞–ø—Ä–∞–≤–ª—è–µ—Ç **–≤—ã–≤–æ–¥** –ø–µ—Ä–≤–æ–π –∫–æ–º–∞–Ω–¥—ã –Ω–∞ **–≤–≤–æ–¥** –≤—Ç–æ—Ä–æ–π (–∫–æ–Ω–≤–µ–π–µ—Ä)  
   - **–ü—Ä–∏–º–µ—Ä**:  
     ```bash
     ls -l | grep ".txt"  # –ù–∞–π–¥–µ—Ç –≤—Å–µ txt-—Ñ–∞–π–ª—ã
     ```

---

### –ü–æ–ª–µ–∑–Ω–∞—è —à–ø–∞—Ä–≥–∞–ª–∫–∞

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ü—Ä–∏–º–µ—Ä                | –ê–Ω–∞–ª–æ–≥ –Ω–∞ —Ä—É—Å—Å–∫–æ–º             |
|----------|-----------------------|--------------------------------|
| `&`      | `cmd &`               | "–°–¥–µ–ª–∞–π —ç—Ç–æ –≤ —Ñ–æ–Ω–µ"           |
| `&&`     | `cmd1 && cmd2`        | "–°–¥–µ–ª–∞–π cmd2, –µ—Å–ª–∏ cmd1 —É–¥–∞–ª—Å—è" |
| `||`     | `cmd1 || cmd2`        | "–°–¥–µ–ª–∞–π cmd2, –µ—Å–ª–∏ cmd1 –ø—Ä–æ–≤–∞–ª–∏–ª—Å—è" |
| `;`      | `cmd1; cmd2`          | "–°–¥–µ–ª–∞–π cmd1, –ø–æ—Ç–æ–º cmd2"      |
| `|`      | `cmd1 | cmd2`         | "–ü–µ—Ä–µ–¥–∞–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç cmd1 –≤ cmd2" |

### –ü—Ä–∏–º–µ—Ä –∏–∑ –≤–∞—à–µ–≥–æ —Å–∫—Ä–∏–ø—Ç–∞
```bash

```

**–ë–µ–∑ `&`** —Å–∫—Ä–∏–ø—Ç "–∑–∞–≤–∏—Å–Ω–µ—Ç", –ø–æ–∫–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –Ω–µ –∑–∞–∫—Ä–æ–µ—Ç –æ–∫–Ω–æ —Ñ–∞–π–ª–æ–≤–æ–≥–æ –º–µ–Ω–µ–¥–∂–µ—Ä–∞.  
**–° `&`** —Å–∫—Ä–∏–ø—Ç –º–≥–Ω–æ–≤–µ–Ω–Ω–æ –ø—Ä–æ–¥–æ–ª–∂–∏—Ç —Ä–∞–±–æ—Ç—É (–µ—Å–ª–∏ —ç—Ç–æ –Ω—É–∂–Ω–æ).

### –ö–æ–≥–¥–∞ —á—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å?
- **`&`** ‚Äì –¥–ª—è –Ω–µ–±–ª–æ–∫–∏—Ä—É—é—â–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π (–æ—Ç–∫—Ä—ã—Ç–∏–µ –æ–∫–æ–Ω, —Å–µ—Ä–≤–∏—Å–æ–≤)
